Exception Handling in JavaScript

Exception handling is a mechanism to handle runtime errors in a controlled manner. It allows developers to write code that can gracefully handle errors without crashing the entire application.

In JavaScript, we can use try-catch blocks to handle exceptions.

        try {
            // code that may throw an error
        } catch (error) {
            // code to handle the error
        } finally {
            // code that will always execute, regardless of whether an error occurred or not
        }
